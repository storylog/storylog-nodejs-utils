# Storylog nodejs utils

## Installation
```sh
npm install https://github.com/storylog/storylog-nodejs-utils.git --save
```

Or using yarn

```sh
yarn add https://github.com/storylog/storylog-nodejs-utils.git
```

## Utilities
- `convertToArray`
- `defaults`
- `filterNil`
- `filterNnull`
- `filterUndefined`

## Usage
```js
import { convertToArray } from 'storylog-nodejs-utils'
```
